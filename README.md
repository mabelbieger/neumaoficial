# Neuma - Plataforma VARK de Aprendizagem Personalizada

Neuma é uma plataforma educacional que utiliza a metodologia VARK para identificar os estilos de aprendizagem dos estudantes e oferecer experiências de estudo personalizadas. A plataforma conecta alunos e professores em um ambiente organizado, permitindo a criação de atividades adaptadas aos diferentes perfis de aprendizagem.

## Sobre o Projeto

A plataforma aplica um teste VARK composto por 10 perguntas para identificar o estilo predominante de aprendizagem de cada estudante. Com base nos resultados, são fornecidas recomendações de estudo personalizadas e atividades adequadas ao perfil identificado.

Os professores podem criar turmas, gerenciar alunos e desenvolver conteúdos específicos para cada estilo de aprendizagem, tornando o processo educacional mais inclusivo e eficiente.

## Funcionalidades

### Estudantes

* Cadastro e autenticação de usuários
* Realização do teste VARK com 10 perguntas
* Identificação do estilo de aprendizagem predominante
* Análise dos perfis:

  * Visual
  * Auditivo
  * Leitura e Escrita
  * Cinestésico
* Recebimento de dicas de estudo personalizadas
* Entrada em turmas através de código de acesso
* Visualização de atividades disponibilizadas pelos professores

### Professores

* Criação e gerenciamento de turmas
* Geração automática de códigos de acesso para turmas
* Gerenciamento de alunos participantes
* Criação de atividades personalizadas por estilo de aprendizagem
* Organização de conteúdos por tipo sensorial
* Dashboard para acompanhamento das atividades das turmas

## Tecnologias Utilizadas

* CSS

  * Authentication
  * Database
* Lucide React

## Objetivo

O objetivo do Neuma é promover uma aprendizagem mais eficiente e personalizada, respeitando as características individuais de cada estudante e fornecendo aos professores ferramentas para adaptar seus métodos de ensino aos diferentes perfis de aprendizagem.
